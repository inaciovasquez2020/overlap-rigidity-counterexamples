# Overlap Rigidity Counterexamples

This repository contains the counterexample constructions for **Overlap Rigidity**
used in the Final Wall program.

## Contents

- `*.tex` — LaTeX expositions of counterexamples
- `*.pdf` — compiled outputs (not tracked)
- `src/` — supporting source (if any)
